# 🎨 Опросник: Отношение к нейросетям в творчестве

## 📌 О проекте
Веб-приложение для сбора и анализа мнений о использовании ИИ в творческой работе.

## 🚀 Как запустить локально
1. Установите зависимости: `pip install -r requirements.txt`
2. Положите `serviceAccountKey.json` в эту папку
3. Запустите: `streamlit run app.py`

## 🌐 Онлайн-версия
Приложение задеплоено на Streamlit Community Cloud.
